BankLite — Mobile Banking UI

Secure onboarding, intuitive transfers, and real-time insights for mobile banking users.

Overview

BankLite is a mobile-first banking interface designed for clarity, speed, and trust. I led the UI/UX design, focusing on secure onboarding, frictionless transfers, and real-time account insights. The design balances simplicity with robust functionality.

Key features

  • Biometric login and secure onboarding
  • One-tap transfers and bill payments
  • Real-time transaction feed and analytics

Role & stack

  • Role: UI/UX Design
  • Stack: Figma, Design tokens, Accessibility audit
  • Timeline: 5 weeks
  • Prototype: View
Finance Security Mobile UX

Outcome

User testing showed a 30% faster onboarding process and improved confidence in transaction flows. The design passed accessibility checks and performed well across devices.
